# Virtual Closet Manager

A Java-based desktop application developed as a **Second Semester Object-Oriented Programming (OOP)** project. The application helps users organize clothing items, manage outfits, and receive clothing recommendations through a simple graphical user interface.

---

## Project Overview

The Virtual Closet Manager allows users to maintain a digital wardrobe by categorizing different clothing items such as shirts, pants, and shoes. It demonstrates the practical implementation of Object-Oriented Programming concepts while providing an easy-to-use Java Swing GUI.

---

## Features

-  Manage clothing items
-  Store shirts, pants, and shoes separately
-  Create and organize outfits
-  Clothing recommendation system
-  User-friendly Java Swing GUI
-  Organized object-oriented project structure

---

## How to Run

1. Open the project in NetBeans IDE.
2. Build the project.
3. Run `MainGUI.java`.
4. Explore the Virtual Closet Manager application.
